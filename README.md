# Solidity Gas优化专题

### 整体原则

Gas消耗场景：部署合约，调用合约（包含读和写）。

Gas消耗大小：
- 写操作 > 读操作
- 存储 > 内存 > 字节码
- 链上交互 > 链下交互
- 运行成本 >> 部署成本
- 高频交互 >> 低频交互
- 用户交互 >> 管理员交互

Gas廉价操作：
- 读写内存变量
- 读取常量和不可变变量
- 读取和写入局部变量
- 读取 calldata 变量，例如 calldata 数组和结构
- 内部函数调用
  
Gas昂贵操作：
- 读取和写入存储变量（状态变量）
- 外部函数调用
- 循环
